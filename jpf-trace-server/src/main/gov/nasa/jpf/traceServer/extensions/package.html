Provides various extensions for Trace Server, to show user how to extend different parts of the system.

It contains instances of {@link gov.nasa.jpf.traceEmitter.TraceEmitter}, {@link gov.nasa.jpf.traceEmitter.traceFilter.TraceFilter}, {@link gov.nasa.jpf.traceServer.printer.EventPrinter}, {@link gov.nasa.jpf.traceServer.traceStorer.PropertyID}, that shows how to store events other than the ones stored by the emitters from {@link gov.nasa.jpf.traceEmitter}, how to augment the properties of the event, and how to prints those properties as a part of the trace.
