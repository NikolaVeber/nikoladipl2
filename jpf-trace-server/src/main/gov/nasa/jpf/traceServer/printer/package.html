Provides classes for printing the trace and for printing the individual events.

Defines two printers that extend {@link gov.nasa.jpf.report.ConsolePublisher}:

  • {@link gov.nasa.jpf.traceServer.printer.ConsoleTracePrinter} - trace printer that mimics the default JPF console printer. One can augment the trace output by adding the {@link gov.nasa.jpf.traceEmitter.PropertyCollection#TRACE_EXTRA_DATA} property to event objects. It will be printed at the end of each event output.
  • {@link gov.nasa.jpf.traceServer.printer.GenericConsoleTracePrinter} - Console printer parameterized with {@link gov.nasa.jpf.traceServer.printer.TracePrinter}.

{@link gov.nasa.jpf.traceServer.printer.EventPrinter} defines the interface for printing individual events. Default implementations are provided for each {@link gov.nasa.jpf.traceServer.traceStorer.EventTypes.eventGroupType}.

{@link gov.nasa.jpf.traceServer.printer.TracePrinter} prints events by using EventPrinter implementations. Custom printers can replace the defaults ones (see {@link gov.nasa.jpf.traceServer.extensions}). TracePrinter can be configured to print or not to print certain event group. In addition to using the TracePrinter in the GenericConsoleTracePrinter, the TracePrinter can also be used in the analyzers (see {@link gov.nasa.jpf.traceAnalyzer.ExecTrackerAnalyzer}).
